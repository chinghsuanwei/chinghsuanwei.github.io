I was a little surprised, There has only one question.

Q1.
There has 3 rules for a legal string

  1. a string must start with a consonant
  2. no two consecutive vowels
  3. no two consecutive consonants

Just consider only upper-case

At first time, I just wanted to use bruteforce-way to solve the problem. But I thought I could not pass the large test if I used bruteforce-way.
Then I spend like 20-min to figure out the question, And found out that it’s just a permutations problem.

The count of vowels must equal or less than 1 to consonants

So for alphabets “AEBBC”, there has 2! * (3! / 2!) = 6 kind of permutations

  1. “BABEC”
  2. “BACEB”
  3. “CABEB”
  4. “CEBAB”
  5. “BECAB”
  6. “BEBAC”

Set Up Hexo

發表於 2018-09-23 更新於 2020-05-09

問題:
我設定_config.yml的git配置如下

deploy:
type: git
repo: https://github.com/chinghsuanwei/chinghsuanwei.github.io.git
branch: master

是正確的

結果是整個hexo原始碼丟到repository上…

原因:
好像是我自己耍87…
我先將hexo init 產生出來的根目錄整個先用git傳到[user].github.io
可能是這個原因 導致deploy的時候 他不是將 .deploy_git push上去

解法:
我自己是將整個[user].github.io的repository砍掉重建

問題:
Deployer not found: git

解法: npm install hexo-deployer-git –save

若還是沒解掉 就可能像我一樣 之前安裝有錯誤(沒下–save?)

解法: 先移除 再裝一次就解了
npm remove hexo-deployer-git
npm install hexo-deployer-git –save

Extra:
若使用git有問題 可能是

  1. 你沒裝git
  2. 盡量使用git bash 別用cmd or PowerShell

Macbook Pro 2010 mid

發表於 2017-05-08 更新於 2020-05-09 分類於 Hardware Setting

If you are using Macbook pro 2010 mid, remember to install gfxCardStatus and turn on 僅Integrated晶片.
Otherwise your computer will crash often when OS decide to use external graphic card.
Reference: https://discussions.apple.com/thread/4766577?start=4695&tstart=0

Notice! After upgrade to 10.12, the gfxCardStatus 2.3 version won’t work.
I need to use previous version like v1.7.5, to forcely turn on Internal Graphic Card only mode.
