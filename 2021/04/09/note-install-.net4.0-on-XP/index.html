Install .Net Framework 4.0 on Embedded Windows XP

發表於 2021-04-09 更新於 2021-04-10 分類於 Note

下載安裝檔

https://dotnet.microsoft.com/download/dotnet-framework/net40

通常Embedded Windows 的網路不會很好(或著會有安全連線的問題?)，所以我們選擇下載Offline Installer

安裝結束後跑出 Installation Did Not Succeed的視窗，並顯示Fatal error during installation.，此時可以點log file看一下log，打開log可能會失敗，若失敗則先按Finish，可以在C:\Documents and Settings\[Username]\Local Settings\Temp 找到log檔案。

我自己安裝是遇到 Error 2769

發現問題是 EWF的關係
該系統的C槽是有防寫入機制，所以要先將其關閉
所以需要先將EWF Write Protection 關閉
打開cmd輸入

並重開機
重開機後可以打開cmd輸入

看看State是否為Disable
之後安裝應該就會成功，安裝好之後再將EWF Write Protection 開啟
打開cmd輸入

並再次重開機

但不知道為什麼 執行.Net Framework 3.5 還是 4.0的程式
還是會丟出

MissingMethodException
Exception Message: method not found: “Boolean System. Runtime. Serialization. DataContractAttribute. get_IsReference ()”.

會有這個Exception的原因 是因為我們的程式有使用JsonSerializer
但是我們沒有加入DataConstract 這個Annotation
因為從3.5 還是 4.0 的.Net Framework public class member 會自動處理Json Serialize的問題
最後還是要在安裝 .Net Framework 3.5版本才能正常的運行
