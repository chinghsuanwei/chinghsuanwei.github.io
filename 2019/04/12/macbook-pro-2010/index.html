Macbook Pro 2010 mid

If you are using Macbook pro 2010 mid, remember to install gfxCardStatus and turn on 僅Integrated晶片. Otherwise your computer will crash often when OS decide to use external graphic card.
Reference: https://discussions.apple.com/thread/4766577?start=4695&tstart=0

Notice! After upgrade to 10.12, the gfxCardStatus 2.3 version won’t work. I need to use previous version like v1.7.5, to force turning on Internal Graphic Card only mode.
